union_native
==========

> Cross-Platform script driven application framework.

## Core Components
- Native window management. [`GLFW3`]
- Render system. [`OpenGL`/`WebGPU`/`Metal`]
- Event loop [`libuv`]

## Graphics API
|Platform/GraphicsAPI|GLES3/WebGL2|WebGPU|Metal|
|:----:|:------:|:---:|:---:|
|Window|complete|  WIP|     |
| MacOS|complete|  WIP|  WIP|
| Linux|complete|  WIP|     |
