union_native
==========

> Cross-Platform script driven application framework.

## Core Components
- Native window management.
- Script graphics api. [`webgpu`]
- Event loop [`libuv`]
- Data Storage [`leveldb`]

## Graphics API
- [`complete`] Metal on MacOS.
- [`WIP`] D3D12 on Windows.
- [`WIP`] vulkan on Linux.

## Install
```bash
npm install @union_native/un
```
